目前專案目標
------------
原本這個 repository 是「台股每日自動選股」專案。這次迭代後，使用者要求移除原本 GitHub 上的每日選股程式，只保留每日新聞 Telegram 推送功能。

目前功能：
- 每天台北時間 08:00 自動執行 GitHub Actions。
- 抓取新聞 RSS。
- 透過 Telegram 發送新聞標題與連結。
- 目前只發送兩個分類：
  - 國際：最近 10 則
  - 財經：最近 10 則
- 訊息內容只包含標題和連結，不包含摘要、來源、時間。

使用者偏好與決策
----------------
1. 一開始使用者想要每日新聞更新發送功能。
2. 先建立過 Codex thread 內的每日新聞自動化，但後續改成整合到既有 GitHub/Telegram 專案。
3. 使用者希望發送到 Telegram。
4. 原本新聞格式包含台灣、國際、科技、財經與摘要；使用者收到測試訊息後要求簡化。
5. 最終定版格式：
   - 只要財經、國際
   - 最近新聞前 10 條
   - 只保留標題
   - 每條附上連結
6. 使用者後來要求移除原本 GitHub 的每日選股程式，因此 repository 現在應視為「每日新聞 Telegram 推送」專案。
